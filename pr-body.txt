### Description

Implement the Delta-Selected Entry + LTP-Matched Hedge strategy logic for NIFTY, complete with weekly lockout (`done-for-this-week`) flag checks, dynamic 1.1% Stoploss and 1.5% Profit Target MTM monitoring, and Tuesday 16:00 IST auto-clearing.

### Changes

- Modified `src/flags/flagWatcher.ts` to support checking the `done-for-this-week` lockout file.
- Updated `src/index.ts` to suspend WebSocket stream if lockout is active.
- Modified `src/scheduler/cronScheduler.ts` to skip tasks if lockout is active, and added a weekly Tuesday 16:00 IST cron task to auto-clear the flag.
- Refactored `src/strategy/strategyManager.ts` to select T0 strikes within the delta range `[0.10, 0.15]` using live Greeks IV when available, and match T1 hedges within 5% (falling back up to 10% premium with 10-step cap).
- Updated `src/execution/executionManager.ts` to trigger exits at 1.1% SL and 1.5% Target, writing the `done-for-this-week` flag on hit.
- Added comprehensive unit tests in `__tests__/strategyManager.test.ts`, `__tests__/executionManager.test.ts`, `__tests__/cronScheduler.test.ts`, `__tests__/flagWatcher.test.ts`, and `__tests__/brokerClient.test.ts` to ensure 100% code coverage.

### Verification

- `npm run test` completes successfully with 100% statement, branch, line, and function coverage.
- Generated basket manually to verify correct selection of T0 strikes and T1 hedges.
